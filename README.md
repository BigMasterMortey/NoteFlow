# NoteFlow

NoteFlow is a minimalist, skeuomorphic personal note-taking application designed with a "notebook-first" philosophy. It provides a focused writing environment that replicates the tactile feel of a physical journal while offering modern digital management tools.

## Core Features

*   **Skeuomorphic Interface**: A grid-aligned writing experience with a red vertical margin and dynamic horizontal lines.
*   **Focus Mode**: A distraction-free environment that slides away UI elements to prioritize writing.
*   **Rich Text Support**: Integrated floating toolbar for bolding, underlining, and highlighting content.
*   **Note Security**: Built-in PIN locking mechanism for private entries.
*   **Organization**: Search and filter functionality to manage notes by date or priority (starred).
*   **Persistence**: Automatic data preservation using browser local storage.
*   **Night Study Mode**: High-contrast dark theme with adaptive notebook lines for reduced eye strain.

## Technology Stack

*   **Frontend**: HTML5, CSS3 (Custom Variables, Flexbox, Transitions)
*   **Logic**: Vanilla JavaScript (ES6+)
*   **Storage**: Browser LocalStorage API

## Getting Started

### Prerequisites
A modern web browser (Chrome, Firefox, Safari, or Edge).

### Local Execution
1. Clone the repository:
   ```bash
   git clone https://github.com/BigMasterMortey/NoteFlow.git
   ```
2. Open `index.html` in your preferred browser.
